🚀 Project: DEEP SCAN

"전 세계 뉴스를 지켜보는 AI, 우리 회사의 24시간 전략실"

복잡한 뉴스 홍수 속에서 '진짜 중요한 기회'만 찾아냅니다.

SGR AI Strategy Team | 2025.12.22

📋 1. 프로젝트 개요

"뉴스, 너무 많아서 힘드시죠?"

  • Why?: 매일 쏟아지는 수만 건의 뉴스, 언제 다 읽으시겠습니까?
  • What?: AI가 잠도 안 자고 지켜보다가 중요한 신호만 잡아냅니다.
  • How?: 전략 보고서를 자동으로 써줘서 일하는 속도가 10배 빨라집니다.

👥 2. 누가 쓰나요? (사용자)

삼성글로벌리서치(SGR) 전략 분석가

😫 현재의 어려움

  • 아침마다 수십 개 영어 뉴스 사이트 뒤지기
  • 번역하고 요약하느라 시간 다 보냄
  • "그래서 이게 우리한테 왜 중요한데?" 깊은 생각 할 시간이 없음

😍 원하는 것

  • 실시간: 뉴스가 뜨자마자 알기
  • 깊이: 단순 요약 말고 전략적 분석
  • 실천: 당장 뭘 해야 할지 제안받기

🔑 3. 핵심 기능 (어떤 일을 하나요?)

  1. 🔍 뉴스 수집 (Data Collector)
    "HBM", "배터리" 같은 키워드만 주면 전 세계를 뒤져옵니다.
  2. 🧠 SGR 드림팀 분석 (Multi-Agent)
    3명의 전문가 AI (거시경제/기술/전략)가 토론하여 심층 리포트 작성
  3. 🎯 리스크 매트릭스 (Risk Heatmap)
    위험도와 영향도를 차트로 시각화 + 점수 산정 근거 제시

🆚 4. 챗GPT랑 뭐가 다른가요?

"우리는 '채팅'이 아니라 '전문 업무 도구'입니다."

구분 🤖 일반 챗GPT 🚀 Project DEEP SCAN
분석 깊이 단순 요약 3인 전문가(드림팀) 토론 분석
시각화 텍스트 위주 위험도 매트릭스(2x2) 차트 제공
설명력 결과만 통보 왜 위험한지 '근거'를 설명 (XAI)

💻 5. 사용된 기술 (Tech Stack)

구분 기술 이름 역할 (쉽게 말하면)
화면 Streamlit 웹사이트 화면을 만드는 도구
관리 LangChain AI에게 일을 순서대로 시키는 관리자
두뇌 Groq (Llama 3) 현존 가장 빠른 속도의 초고성능 무료 AI
재료 NewsAPI 전 세계 뉴스를 실시간으로 가져오는 곳

🔄 6. 사용 방법 (User Flow)

"딱 3단계면 끝납니다"

  1. 입력 ⌨️
    궁금한 주제 입력 (예: "테슬라 미래 전략")
  2. 처리 ⚙️
    AI가 뉴스를 읽고 고민합니다. (수집 ➡ 분석)
  3. 결과 📑
    전략 보고서가 짠! 하고 나옵니다.

📊 7. 기대 효과 (뭐가 좋아지나요?)

"빠르고, 정확하게"
  • ⚡ 일이 10배 빨라짐
    자료 조사: 3시간 ➡ 1분에 끝!
  • 🎯 판단을 도와줌
    "그냥 이런 일이 있었다"가 아니라 "그래서 이렇게 해야 한다"를 알려줌
  • 🛡️ 위험 대비
    경쟁사의 움직임이나 위험 요소를 누구보다 빨리 발견

🗺️ 8. 앞으로의 계획 (Roadmap)

  • 1단계 (현재) ✅
    뉴스 보고 AI가 분석해주는 기능 완성!
  • 2단계 (확장) 🔜
    • 주식/공시 연동: 주가랑 실적도 같이 봅니다.
    • 자동 알림: 중요한 일 생기면 슬랙/메신저로 알려줍니다.
  • 3단계 (최적화) 🚀
    우리만의 AI: 회사 보안을 위해 우리 회사 전용 AI를 만듭니다.
