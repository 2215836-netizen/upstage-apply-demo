# 🚀 Project: DEEP SCAN
### **"전 세계 뉴스를 지켜보는 AI, 우리 회사의 24시간 전략실"**

**복잡한 뉴스 홍수 속에서 '진짜 중요한 기회'만 찾아냅니다.**

# 📋 1. 프로젝트 개요
**"뉴스, 너무 많아서 힘드시죠?"**

- **Why?**: 매일 쏟아지는 수만 건의 뉴스, **언제 다 읽으시겠습니까?**
- **What?**: AI가 잠도 안 자고 지켜보다가 **중요한 신호**만 잡아냅니다.
- **How?**: **전략 보고서**를 자동으로 써줘서 일하는 속도가 **10배 빨라집니다.**

# 🔑 3. 핵심 기능 (어떤 일을 하나요?)

1.  **🔍 뉴스 수집 (Data Collector)**
    - "HBM", "배터리" 같은 키워드만 주면 전 세계를 뒤져옵니다.
2.  **🧠 전략 분석 (Strategic Analyzer)**
    - 단순 요약 ❌ ➡ **전문가처럼 깊게 분석** ⭕
    - AI가 **기회**와 **위협**을 스스로 생각해서 찾아냅니다.
3.  **📊 결과 보여주기 (Dashboard)**
    - **한 장짜리 보고서**로 깔끔하게 정리해 줍니다.
    - 임원분들께 바로 보여드릴 수 있습니다.

# 🆚 4. 챗GPT랑 뭐가 다른가요?

**"우리는 '채팅'이 아니라 '전문 업무 도구'입니다."**

| 구분 | 🤖 일반 챗GPT | 🚀 Project DEEP SCAN |
| :--- | :--- | :--- |
| **정보** | 학습된 과거 데이터 중심 | **지금 막 뜬 '실시간 뉴스'** |
| **동작** | 내가 일일이 다 물어봐야 함 | **키워드만 넣으면 알아서 완성** |
| **결과** | 그냥 줄글 답변 | **경영진 보고용 SWOT 보고서** |

# 💻 5. 사용된 기술 (Tech Stack)

| 구분 | 기술 이름 | 역할 (쉽게 말하면) |
| :--- | :--- | :--- |
| **화면** | `Streamlit` | 웹사이트 화면을 만드는 도구 |
| **관리** | `LangChain` | AI에게 일을 순서대로 시키는 관리자 |
| **두뇌** | `OpenAI API` | 뉴스를 읽고 분석하는 **똑똑한 AI** |
| **재료** | `NewsAPI` | 전 세계 뉴스를 실시간으로 가져오는 곳 |

# 🔄 6. 사용 방법 (User Flow)
**"딱 3단계면 끝납니다"**

1.  **입력** ⌨️
    - 궁금한 주제 입력 (예: "*테슬라 미래 전략*")
2.  **처리** ⚙️
    - AI가 뉴스를 읽고 고민합니다. (수집 ➡ 분석)
3.  **결과** 📑
    - **전략 보고서**가 짠! 하고 나옵니다.
